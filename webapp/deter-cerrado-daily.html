Informação de alteração nos dados
O limite dos biomas brasileiros foram alterados conforme publicação do IBGE de 30/10/2019. Os dados de supressão e/ou degradação de vegetação publicados na plataforma TerraBrasilis foram ajustados para o novo recorte em fevereiro de 2022. Acesse aqui os resultados congelados até o dia 31 de Dezembro de 2021, relativos ao antigo limite.
Imprimir Gráficos
A impressão deste documento utiliza os recursos do navegador e pode apresentar diferenças entre eles.
A impressão acomoda bem os gráficos quando as configurações são as seguintes:
   - A orientação do papel é do tipo paisagem;
   - O tamanho de papel é A4;
   - E margens padrão do navegador.

Sobre TerraBrasillis

O TerraBrasilis é uma plataforma desenvolvida pelo INPE para organização, acesso e uso através de um portal web dos dados geográficos produzidos pelos seus programas de monitoramento ambiental.
